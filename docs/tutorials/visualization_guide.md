# Руководство по визуализации

В папке `gra_ramanujan/visualization/` находятся функции для построения графиков на основе логов.

## Функции

### `plots.plot_evolution(csv_path)`
Строит график максимальной красоты формул по поколениям для каждой константы. Позволяет увидеть, как улучшается качество находок.

### `plots.plot_heatmap(csv_path)`
Теплокарта: поколения по горизонтали, константы по вертикали, цвет – максимальная красота. Помогает выявить, по каким константам прогресс идёт быстрее.

### `plots.plot_complexity_distribution(csv_path)`
Ящик с усами (boxplot) распределения сложности формул по поколениям. Показывает, становится ли популяция склонна к более простым или более сложным выражениям.

## Дополнительно

В `gallery.py` есть функция `generate_latex_gallery(csv_path, output_file)`, которая выбирает лучшие формулы для каждой константы и записывает их в LaTeX-файл для отчёта.
